# git 

git is version control


## command 

```
// for initializing git
// only one time for each repo
git init

//for adding 
git add <filename>/<foldername> // eg. git add file.txt, git add .

// git commit 
git commit -m "your expected message"

// to know status
git status

// to know difference 
git diff 

kichu likhlam

```
